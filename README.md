This script asks the user details about which state, district, assembly, language, type of roll so that the required pdfs are downloaded. You can even select what range of pdfs you want.

There is a captcha that needs to be solved. It is used to prevent bots but Google's Gemini LLM (2.5-Pro) comes to the rescue by solving the captcha and returning the text.
